* 性能相关

* 窗口相关
* 媒体相关

* 重力感应相关

## 启动服务

### 拉取项目

``git clone https://github.com/tyua07/FP-Browser-Detect.git``

### 设置镜像

``npm config set registry https://registry.npm.taobao.org``

### 安装 依赖

* ``cd FP-Browser-Detect``
* ``npm install``

### 启动服务

``npm run start``
